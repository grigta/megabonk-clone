# MEGABONK — three.js clone

A 3D survivor-like roguelike built in pure HTML + Three.js (no build step). Inspired by **Megabonk**.

**▶ Play:** open `index.html` (served over HTTP) or visit the GitHub Pages link.

## Controls
- **WASD / Arrows** — move
- **Mouse** — turn the camera (click the scene to capture, Esc to release)
- **Q / E** — turn the camera (no mouse capture)
- **Space** — dash
- **P** — pause
- **1 / 2 / 3** — pick a level-up card

## Gameplay
Weapons fire automatically — you just position. Kill enemies, grab the XP gems they drop, level up, and pick from 3 random upgrade cards (weapons + passives with rarity tiers). Survive the 10-minute run; bosses appear over time. Snowball your build.

Three.js is loaded from a CDN via import map, so an internet connection is required.
